# Alien State University of People and Stuff
This repository contains a game being developed as an ADS lab. It's not of any value, just fun.
The game is only available for Linux (Windows support may appear in the future).
Binaries can be found in the releases section, or built from source code using Cmake.

    cd Alien State University of People and Stuff
    ./scripts/makeandrun.sh
    
Unfortunately, the game will continue to be developed. Fortunately, this will only happen as the developer has the time and desire.
Have a good game, take care of your eyes!
